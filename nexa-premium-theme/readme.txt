== Description ==

Nexa Premium is a high-end WordPress theme designed specifically for the Nexa Growth Agency. It features ultra-modern aesthetics, glassmorphism UI elements, and high-performance GSAP animations.

== Installation ==

1. Zip the `nexa-premium-theme` folder.
2. In your WordPress Dashboard, go to Appearance > Themes > Add New.
3. Click "Upload Theme" and select the `nexa-premium-theme.zip` file.
4. Click "Install Now" and then "Activate".
5. Go to Appearance > Menus and create a menu, assigning it to the "Primary" location.

== Features ==

- Dark Mode by default.
- Custom Landing Page via index.php.
- Responsive design for Mobile, Tablet, and Desktop.
- Lucide Icon integration.
- GSAP ScrollTrigger animations.
